# MouseGestureStudio 营销介绍

## 一句话介绍

MouseGestureStudio 是一款面向 Windows 的现代鼠标手势自动化工具：画一个手势，就能执行复制、粘贴、前进后退、打开网页、插入日期、批量处理链接，甚至运行 AI 生成的 JSON 组合动作。

## 核心卖点

- 鼠标手势即自动化入口：按住触发键画出轨迹，松开即执行动作，不需要记一堆快捷键。
- 原生 JSON 组合动作：工具本身是一个“动作运行壳”，用户可以让 AI 生成符合规则的 JSON，粘贴或导入后马上变成自己的自动化能力。
- 不依赖第三方会员功能：内置选择器、日期格式、快捷键、文本写入、批量打开网页、选中文本提取等常用自动化步骤。
- 面向真实文本环境：批量打开网址时会自动清洗空行、说明文字、标点和重复链接，单个网址失败也不会中断后续动作。
- 可视化动作库：每个动作都有轨迹预览、方向箭头、动作类型标签，新手也能看懂当前手势会做什么。
- 动作存档可迁移：支持导入/导出动作存档，换电脑或分享动作库都更轻松。

## 适合谁使用

- 每天重复复制、粘贴、打开网页、整理链接的办公用户。
- 需要快速插入日期、日报标题、固定文本模板的内容创作者。
- 希望用 AI 生成自动化流程，但不想学习脚本语言的新手用户。
- 喜欢鼠标手势、快捷启动和轻量自动化的效率工具用户。

## 使用场景示例

- 画一个向左手势：浏览器后退。
- 画一个向上手势：复制当前选中内容。
- 画一个折线手势：弹出日期格式选择框，选择后自动写入输入框。
- 画一个 Z 字手势：一次打开每天固定要看的网页。
- 选中一段混杂文本后画手势：自动提取里面的网址并逐个打开。
- 让 AI 生成一个“早报标题选择器”JSON：导入后立刻变成自己的专属动作。
