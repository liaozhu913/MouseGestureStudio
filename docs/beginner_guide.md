# MouseGestureStudio 新手使用教程

## 1. 启动软件

打开：

```text
D:\MouseGestureStudio\dist\MouseGestureStudio\MouseGestureStudio.exe
```

启动后会看到主界面。关闭窗口时，程序会最小化到系统托盘；如果要完全退出，请右键托盘图标选择“退出”。

## 2. 理解主界面

主界面分为三块：

- 顶部区域：显示软件名称、当前启用动作数量和触发键。
- 控制区域：可以启用/停用手势、选择触发键、导入/导出动作存档、添加新手势。
- 动作卡片区：每张卡片代表一个手势动作，卡片上能看到轨迹预览、动作类型、动作名称和提示。

## 3. 第一次使用手势

默认触发键是鼠标右键。

1. 保持“默认启用手势”为勾选状态。
2. 按住鼠标右键不要松开。
3. 按照卡片里的轨迹方向移动鼠标。
4. 松开右键。
5. 如果轨迹匹配成功，软件会执行对应动作。

常见默认动作：

- 向左画：浏览器后退。
- 向右画：浏览器前进。
- 向上画：复制。
- 向下画：粘贴。
- 斜向左下：截图。

## 4. 编辑一个动作

1. 在主界面双击某张动作卡片。
2. 在“操作名称”里修改名称。
3. 在“操作类型”里选择动作类型。
4. 根据类型填写内容，例如快捷键、文本、网址或组合动作 JSON。
5. 点击“保存”。
6. 回到主界面后点击“应用设置”。

## 5. 新建一个手势

1. 点击“添加手势轨迹”。
2. 在弹出的轨迹窗口里按提示绘制一个新手势。
3. 给手势设置动作。
4. 保存后，新的动作卡片会出现在主界面。

建议新手先画简单、区别明显的形状，例如直线、折线、L 形、Z 形。

## 6. 使用内置 JSON 组合动作

软件已经内置了几个组合动作：

- 选择日期格式：弹出日期格式选择框，选择后自动粘贴。
- 日报日期选择器：弹出自定义日报日期选项。
- 打开每日网页：一次打开多个固定网页。
- 打开选中网址：复制当前选中文本，自动提取里面的网址并逐个打开。

使用“打开选中网址”的方法：

1. 在网页、文档或聊天窗口中选中一段包含网址的文本。
2. 画出“打开选中网址”对应的手势。
3. 软件会自动复制选中文本，清洗出真正的网址并打开。

它可以处理空行、普通说明文字、中文标点、英文标点和重复网址。

## 7. 让 AI 帮你生成动作

如果你想做自己的组合动作，可以打开：

```text
D:\MouseGestureStudio\docs\ai_workflow_prompt_template.txt
```

把里面的提示词发给 AI，然后把你的需求补充进去。比如：

```text
我的需求：
每天早上打开公司后台、日报页面和日历页面，每个网页间隔 300 毫秒。
```

AI 会输出一段 JSON。然后：

1. 双击一个手势卡片进入编辑。
2. 操作类型选择“组合动作”。
3. 把 AI 生成的 JSON 粘贴到“步骤 JSON”。
4. 点击“格式化/校验”。
5. 校验通过后保存。

## 8. 导入和导出动作存档

如果你想备份或迁移动作库：

- 点击“导出存档”，保存当前全部动作。
- 点击“导入存档”，选择之前导出的 JSON 文件。
- 导入时可以选择合并到当前动作库，也可以替换当前全部动作。

建议在大量修改动作前先导出一份存档。

## 9. 常见问题

### 按住右键没有画线

请确认“默认启用手势”已经勾选。如果仍无反应，尝试退出软件后重新启动。

### 手势识别不准

请进入对应动作，点击“重绘轨迹”，用更稳定、明显的形状重新录入。

### 批量打开网址时夹杂了普通文字

不用手动清理。使用内置的 `open_urls` 动作时，软件会自动提取文本里的网址。

### 关闭窗口后程序还在运行

这是正常行为。程序会进入系统托盘，方便继续监听手势。如果要退出，请右键托盘图标选择“退出”。
